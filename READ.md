# Personal Portfolio Website

A responsive personal portfolio website built using HTML and CSS to showcase my profile, skills, education, projects, and achievements.

## 👨‍💻 About

This portfolio website represents my academic background, technical skills, projects, and achievements as a Computer Science Engineering student.

- Professional Portfolio Design
- About Me Section
- Technical Skills Section

- Project Showcase
- Achievements & Certificates

## 💻 Installation

1. Clone the repository:

```bash

```

2. Open the project folder:

```bash

```

3. Run the website by opening:

